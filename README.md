# Crumblr

Lars Kellynn, Software Development 1, 202620-CEN-3024C-23585

Introduction: Crumblr is a bakery management system, used by cafe employees to track their menu items. Employees will be able to
track, create, edit and delete bakery items from the system. They will also be able to print a list of the current inventory items and their attributes. These include: the item’s description, item quantity, the date it was created, its shelf life, the expiration date, and any known allergens. A custom action of calculating the menu item's expiration date will be done automatically by the system.

Requirements Definition: Users will interact with the application using the CLI with no GUI. Data will be stored locally and inputted via a text file from a path entered by the user. Output will be printed to the CLI.
